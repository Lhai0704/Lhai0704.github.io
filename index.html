福利经济学：研究资源配置如何影响经济福利。

市场上的供求均衡可以最大化买者和卖者得到的总利益。

消费者剩余

支付意愿

一个买者愿意支付的最高价格。

消费者剩余 = 愿意支付的量 - 实际支付的量

用需求曲线衡量消费者剩余

需求曲线以下和价格以上部分的面积衡量一个市场上的消费者剩余。

价格降低如何增加消费者剩余

生产者剩余

市场效率

仁慈的社会计划者

总剩余 = 买者的评价 - 卖者的成本

效率：资源配置使社会所有成员得到的总剩余最大化。

市场均衡的评价

自由市场是组织经济活动的最好方法。

结论：市场效率与市场失灵

福利经济学的基本工具：消费者剩余与生产者剩余。

最重要的两个假设：

  1. 市场是完全竞争的。单一卖者或买者可以控制市场价格。这种影响价格的能力称为市场势力。
  2. 市场结果只影响参与市场的买者和卖者。外部性（污染）。

在存在市场势力或外部性等市场失灵的情况下，市场不能有效地配置资源。

供给、需求与政府政策

发表于 2020-09-30 | | 阅读次数:

《经济学原理 微观经济学》 第六章 供给、需求与政府政策

价格控制

价格上限如何影响市场结果

当政府对竞争市场实行限制性价格上限时，产生物品的短缺，卖者在大量潜在买者中配给稀缺物品。

价格下限如何影响市场结果

限制性价格下限引起了过剩。

对价格控制的评价

价格是隐藏在供给曲线和需求曲线背后的千百万企业和消费者决策的结果。

税收

税收归宿：税收负担在市场参与者之间进行分配的方式。

向卖者征税如何影响市场结果

买者与卖者分摊了税收负担。当对一种物品征税时，物品在新均衡时的销售量减少了。

向买者征税如何影响市场结果

对买者征税和对卖者征税是相同的。唯一区别是谁来把钱交给政府。

税收归宿取决于供给和需求的力量。

弹性与税收归宿

税收负担更多地落在缺乏弹性的市场一方身上。不愿意离开市场，承担更多的税收负担。

结论

价格上限短缺，价格下限过剩。

税收缩小市场，买者与卖者共同交税。

弹性及其应用

发表于 2020-09-30 | | 阅读次数:

《经济学原理 微观经济学》 第五章 弹性及其应用

弹性衡量买者与卖者对市场条件变化的反应程度。

需求弹性

需求价格弹性及其决定因素

需求价格弹性：衡量一种物品需求量对其价格变动反映程度的指标。

如果需求量对价格变动反应很大，就说物品的需求是富有弹性的。

  • 相近替代品的可获得性：有相近替代品，富有弹性。
  • 必需品和奢侈品：衡量是否是奢侈品取决于买者的偏好。必须品缺乏弹性，奢侈品富有弹性。
  • 市场的定义：狭窄定义的市场弹性往往大于宽泛定义的市场的弹性。（狭窄定义的物品更容易找到替代品）
  • 时间范围：物品的需求往往在长期内更富有弹性。

需求弹性价格的计算

需求价格弹性=需求量变动% / 价格变动%

中点法

计算百分比时除中点值。

各种需求曲线

弹性大于1，富有弹性。

通过某一点的需求曲线越平坦，需求价格弹性就越大。

总收益与需求价格弹性

总收益取决于需求价格弹性。

缺乏弹性，价格上升引起总收益增加。

沿着一条线性需求曲线的弹性和总收益

在价格低而数量高的各点上，需求曲线是缺乏弹性的。在价格高而数量低的各点上，需求曲线是富有弹性的。

其他需求弹性

需求收入弹性：收入变动时需求量如何变动。

需求的交叉价格弹性：一种物品需求对另一个物品价格变化的反应程度。（正负数取决于替代品还是互补品）

供给弹性

供给、需求和弹性的三个应用

农业的好消息可能对农民来说是坏消息吗

总收益取决于需求弹性。小麦缺乏弹性。供给增加，价格降低，总收益减少。

为什么石油输出国组织不能保持石油的高价格

长期富有弹性，减少产量，价格提高少。

禁毒增加还是减少了与毒品相关的犯罪

结论

Q：在一个时期内，企业进入和退出一个市场的能力意味着在长期中？

A：供给曲线富有弹性，企业是供给者，进入和退出市场的能力强表示灵活增添和削减生产，富有弹性。

供给与需求的市场力量

发表于 2020-09-30 | | 阅读次数:

《经济学原理 微观经济学》 第四章 供给与需求的市场力量

供给与需求是使市场经济运行的力量。

市场与竞争

什么是市场

市场：由某种物品或服务的买者和卖者组成的一个群体。

什么是竞争

物品或服务的价格和销售量是买者和卖者在市场上相互交易而共同决定的。

竞争市场：有许多买者和卖者，以至于每个人对市场价格的影响都微乎其微。

完全竞争：两个特征：1.物品是完全相同的。2.买者和卖者人数众多。

需求

需求曲线：价格和需求量之间的关系

需求量是买者愿意并且能够购买的物品数量。

需求定理：在其他条件不变时，一种物品的价格上升，对该物品的需求量减少。

需求曲线：向右下方倾斜。

市场需求和个人需求

市场需求是个人需求的总和。

需求曲线的移动

向右移动->需求增加。

很多变量使需求曲线移动：

变量 备注
收入 收入减少，正常物品需求减少，低档物品增加。
相关物品的价格 替代品，互补品。
爱好 喜欢什么多买什么。
预期 对未来的预期影响现在对物品和服务的需求
买者的数量 买者数量增加，总需求量增加

只有当除了用坐标轴表示的变量以外的其他相关变量变动时，曲线才会移动。

供给

供给曲线：价格和供给量之间的关系

市场供给和个人供给

供给曲线的移动

变量 备注
投入品价格 投入品价格提高，供给减少
技术 降低生产成本
预期
卖者的数量

记住是供给曲线移动还是沿着供给曲线移动。

供给与需求的结合

均衡

均衡：供给曲线和需求曲线相交与一点。供给量=需求量

均衡价格有时被称为市场出清价格。

许多买者与卖者的活动都会自发的使市场价格向均衡价格移动。

供求定理：任何一种物品的价格都会自发调整，使该物品的供给和需求达到平衡。

分析均衡变动的三个步骤

  1. 确定事件是使供给曲线移动还是使需求曲线移动。
  2. 确定曲线移动的方向。
  3. 用供求图说明这种移动如何改变均衡价格和均衡数量。

结论：价格如何配置资源

供给与需求共同决定了经济中许多不同物品与服务的价格，价格又是引导资源配置的信号。

在市场经济中，价格是配置稀缺资源的机制。

供给的增加将引起沿着既定需求曲线的变动，这种变动称为需求量的变动。

本章学习了供给与需求的关系，提出了两类曲线，交点为均衡，价格是引导经济决策从而配置稀缺资源的信号。

比较优势：专业化的动力

绝对优势

一个生产者用比另一个生产者更少的投入生产某种物品的能力。

机会成本和比较优势

比较优势：一个生产者以低于另一个生产者的机会成本生产某种物品的能力。

一个人有可能在两种物品的生产上都具有绝对优势，但一个人却不可能在两种物品的生产上都具有比较优势。

《经济学原理 微观经济学》 第二章 像经济学家一样思考

作为科学家的经济学家

科学的本质是科学方法：冷静地建立并检验有关世界如何运行的各种理论。

科学方法：观察、理论、进一步观察

假设的作用

假设可以使复杂的世界简单化，从而使解释这个世界变得更为容易。

经济模型

第一个模型：循环流量图

一个说明货币如何通过市场在家庭和企业之间流通的直观经济模型。

经济简化为由两类决策者：企业和家庭组成。企业用劳动、土地、资本（建筑物和机器）这些投入品来生产物品和服务，这些投入品称为生产要素。家庭则拥有生产要素并消费企业生产的所有物品和服务。

家庭和企业在两类市场上相互交易。在物品与服务市场上，家庭是买者，企业是卖者，家庭购买企业生产的物品和服务。在生产要素市场上，企业是买者，家庭是卖者，家庭向企业提供用于生产物品和服务的投入。

第二个模型：生产可能性边界

生产可能性边界：表示在生产要素和生产技术既定时，一个经济所能生产的产品的数量的各种组合。

斜率可以表示机会成本。

微观经济学和宏观经济学

微观经济学：研究家庭和企业如何做出决策，以及他们如何在市场上相互交易。

宏观经济学：研究整体经济现象，包括通货膨胀、失业、经济增长。

作为政策顾问的经济学家

实证分析与规范分析

实证表述是描述性的。规范表述是规定性的。

经济学家意见分歧的原因

科学判断的不同

价值观的不同

感觉与现实

总结

本章学习了两个经济学模型：循环流量图和生产可能性边界。

Q：用经济可能性边界描述效率的思想

A：如果经济可以利用全部稀缺资源，则这种结果是有效率的。所以在经济可行性边界上的生产是有效率的，以内是无效率的。

关键概念：

  • 循环流量图：货币如何通过市场在家庭和企业中流通。
  • 生产可能性边界：在生产要素和生产技术既定时，所能生产的产品数量的各种组合。
  • 微观经济学：市场和家庭如何做出决策，如何在市场上进行交易。
  • 宏观经济学：整体经济现象。
  • 实证表述：描述性。
  • 规范表述：规范性。

经济学十大原理

发表于 2020-09-29 | | 阅读次数:

《经济学原理 微观经济学》第一章 经济学十大原理

人们如何做出决策

1.人们面临权衡取舍

做出决策就是要求我们在一个目标和另一个目标之间进行权衡取舍。

效率：社会能从其稀缺资源中得到的最大利益。

平等：将利益平均地分配给社会成员。

2.某种东西的成本是为了得到它所放弃的东西

机会成本：为了得到某种东西所必须放弃的东西。

3.理性人考虑边际量

边际变动：对行动计划的微小增量调整。

当且仅当一种行为的边际收益大于边际成本时，理性决策者才会采取这种行为。

4.人们会对激励做出反应

激励：引起一个人做出某种行为的某种东西。

分析市场运行时，激励是至关重要的，价格对消费者和生产者行为的影响对于市场经济如何配置稀缺资源是至关重要的。

分析政策时，不仅应该考虑它的直接影响，还应该考虑它通过激励产生的不太明显的简介影响，如果政策改变了激励，那就会使人们改变自己的行为。

人们如何相互影响

5.贸易可以使每个人的状况都变得更好

贸易可以使每个人从事自己最擅长的活动，通过贸易，可以以较低的成本获得各种各样的物品和服务。

贸易使各国可以专门从事自己最擅长的活动，并享有种类更多的物品和服务。

6.市场通常是组织经济活动的一种好方法

市场经济：当许多企业和家庭在物品与服务市场上相互交易时，通过他们的分散决策配置资源的经济。

价格和利己引导着企业和家庭的决策。

价格就是看不见的手用来指引经济活动的工具。

当政府阻止价格根据供求状况自发调整时，它就限制了看不见的手对组成经济的家庭和企业的决策进行协调的能力。（解释了为什么税收对资源配置有不利的影响）

亚当·斯密：经济参与者受利己心所驱动，而市场上这只看不见的手指引这种利己心去促进总体的经济福利。

7.政府有时可以改善市场结果

只有在政府实施规则并维持对市场经济至关重要的制度时，看不见的手才能施展魔力。市场经济需要实施产权（个人拥有并控制稀缺资源的能力）。

政府干预经济并改变人们自己选择的资源配置的原因有两类：促进效率或促进平等。

市场失灵：市场本身不能有效配置资源的情况。

市场失灵：

  • 外部性：一个人的行为对旁观者福利的影响，例：污染
  • 市场势力：单个经济活动者（小群体）对市场价格有显著影响的能力。

整体经济如何运行

8.一国的生活水平取决于它生产物品与服务的能力

生产率：每单位劳动投入所生产的物品与服务的数量。

几乎所有生活水平的差别都可以归因于各国的生产率的差别。

9.当政府发行了过多货币时，物价上升

通货膨胀：经济中物价总水平上升。

10.社会面临通货膨胀与失业之间的短期权衡取舍

货币注入的短期效应：

  • 货币量增加刺激了社会的整体支出水平，增加了对物品和服务的需求。
  • 需求的增加会引起企业提高物价，同时鼓励企业雇佣更多的工人，从而减少失业。

结论与总结

看不见的手是指：尽管市场参与者是利己的，但自由市场仍能达到合意结果的能力。

第一章讲述了经济学的十大原理，了解到了几个关键概念：机会成本，市场失灵等。

容器类widget与布局类widget都作用于其子widget。布局类一般需要接收一个widget数组（children）。容器类widget一般只需要接收一个子widget（child）。布局类widget是按照一定的排列方式来对其子widget进行排列，容器类只是包含子widget对其添加一些修饰，变换，限制。

填充Padding

给子节点添加填充，类似于边距。

EdgeInsetsGeometry的一个字类：EdgeInsets

  • all(double value)
  • only({left, top, right, bottom})
  • symmetric({vertical, horizontal})
